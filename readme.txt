Visually indicate the current server tier (dev, staging, production, etc.)

== Description ==

This plugin provides a simple visual cue that you are currently viewing a particular environment in your arrangement of server tiers. Currently, the only available indicator is a green admin bar. 

== Installation ==

1. Upload `environment-indicator` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= Where are the options for this plugin? =

There are no configurable options yet. Simply activate this plugin on your development environment.

== Screenshots ==

1. Admin bar background indicates that you are on the dev environment
